# OpenGL-Load-Model

OpenGL Load Wavefront 3D Model 

## How to build:
- Set up OpenGL, GLUT
- Clone this project
`git clone https://github.com/WHKnightZ/OpenGL-Load-Model.git`
- Build with make
`make`
- Or build by g++
`g++ main.cpp -o load_model -lGL -lGLU -lglut`
- And run
`./load_model`
