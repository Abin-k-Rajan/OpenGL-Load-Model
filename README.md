# OpenGL-Load-Model

OpenGL Load Wavefront 3D Model 

## How to build:
- Set up OpenGL, GLUT
- Build with make
`
git clone https://github.com/WHKnightZ/OpenGL-Load-Model.git
make
./load_model
`
